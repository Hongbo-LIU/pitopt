# 01 战略规划 Strategic Planning ：代序言

战略规划在英文中常用的术语为Strategic Planning，但是英文翻译过来时，或许就有不同的翻译表达，例如在MBA智库百科中，就分为三个词条：

>Strategic Planning可能是指：
>* 战略规划（Strategic Planning）是指就是制定组织的长期目标并将其付诸实施，它是一个正式的过程和仪式。一些大企业都有意识
  地对大约50年内的事情  做出规划。
>* 战略计划（Strategic Planning）是企业根据外部环境和内部资源条件而制定的涉及企业管理各方面（包括生产管理、营销管理、财务
  管理、人力资源管理  等）的带有全局性的重大计划。
>* 策略规划（Strategic planning）是指由高阶主管负责规划制定公司使命、组织目标、基本政策及策略，以规范达成该组织目标所需
  的资源使用管理。<sup>[1]</sup>

具体表达的内容有很相似。在这里我们不是要讨论翻译的好与差，想要讨论的是作为一个组织或者企业，对于未来的发展都一个基本的战略规划
（在本文中我们统一称为战略规划）。矿山生产企业也不例外，也需要指定出合适的战略规划，一般在（预）可行性研究阶段完成，重点体现的
内容就是明确的生产能力，相应的设备选型，相应的开采规划，开拓规划，以及配套的人力，财力，物力，维护配套，近几年内对于环保的要求
越来越高，所以相应的水土，防尘，植被和未来的复垦计划也在战略规划层面越来越重要。

其中，对于露天矿山来说境界优化是最重要的战略规划内容之一。本文就露天矿的境界优化展开讨论，希望可以做到抛砖引玉，也希望可以给同仁
一些借鉴意义。

## 3.1 价值模型

>地质块状模型是矿物品位及有关地质特征在矿床中分布情况的离散化表述，主要用于储量计算和地质数据的计算机处理。在最终开采境界的计算机优化设计和开
> 采计划优化中，常常用到另一种块状模型－价值块状模型（简称价值模型）。地质块状模型中每一块的特征值是品位或某一地质特征，价值模型中每一块的特征
> 值则是假设将其采出并处理后能够带来的经济净价值。块的净价值是根据块中所含可利用矿物的品位、开采与处理中各道工序的成本及产品价格计算的。矿床所
> 含矿物的种类不同，矿山企业经营体制和成本管理制度不同，计算净价值时，用到的参数不同。 
> 价值模型是优化方法确定境界的基础。[^4]
